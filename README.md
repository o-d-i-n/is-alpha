# is-alphabet

Checks whether passed string contains alphabets, digits and vocabulory.

## Installation
```bash
npm i is-alphabet --save
```

## Usage
```js
const isAlpha = require('is-alphabet');

if(isAlpha('Some string with english alphabets')) {
  console.log('pure english');
}

```
## Development
```bash
npm i
npm test
```
